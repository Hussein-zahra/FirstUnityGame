1. Install the necessary dependencies.
2. Clone the project:
   ```bash
   git clone <repo-url>
   ```
3. Open and run the project in Unity.

## How It Works

This project is built with Unity and includes several key components:

- **Camera Follow System**: The camera automatically follows the player character, keeping them centered on screen
- **Coin Management**: A coin collection system with visual feedback and material changes
- **Door Teleportation**: Players can interact with doors to teleport to different areas
- **Input System**: Modern input handling for smooth player control
- **Scene Management**: Multiple scenes for different game levels and environments

The game mechanics work together to create an engaging gameplay experience where players collect coins, interact with objects, and navigate through different environments.

## Project Features

- Real-time camera following
- Collectible coin system with animations
- Interactive door mechanics
- Multiple themed environments
- Mobile-friendly controls
